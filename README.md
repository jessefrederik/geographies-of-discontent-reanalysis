# Replication Files

**"Geographies of Discontent: A Reanalysis"**
Jesse Frederik

---

## Quick start

To reproduce the paper from scratch:

```bash
make
```

This runs all R analyses, generates tables and figures, and compiles three PDFs:

- **`size_gradient_report.pdf`** — main paper (~13 pages, the journal-submission edition)
- **`size_gradient_report_appendix.pdf`** — standalone appendix (Online Supporting Information edition)
- **`size_gradient_report_with_appendix.pdf`** — combined edition: main paper followed by the appendices as one document

Compiled PDFs are also tracked in the repo as convenience copies, so the paper can be read directly without running the pipeline. The Makefile also exposes individual targets: `make paper`, `make appendix`, `make with-appendix`.

Requires R (4.4+), the listed R packages, and a LaTeX distribution (TeX Live or MiKTeX). On first run, the 2022 Italian election data (~50MB) and French election data (~200MB) are downloaded automatically. Subsequent runs use cached data. The OpenCivitas and Ministry of the Interior data needed for the mechanism analysis (§4) ship in this repo (`data_raw/italy/`, ~121MB).

## Software requirements

**R** version 4.4.1 (2024-06-14) on macOS (aarch64-apple-darwin20). Required packages:

| Package | Purpose |
|---------|---------|
| haven | Reading Stata `.dta` files |
| fixest | Two-way fixed-effects regressions |
| MatchIt | Mahalanobis matching for MTWFE |
| data.table | Data manipulation |
| rdrobust | Regression discontinuity estimates |
| rddensity | McCrary density test |
| synthdid | Synthetic difference-in-differences |
| ggplot2 | Figures |
| scales | Axis formatting |
| stringi | String normalisation (matching Italian municipality names) |
| patchwork | Multi-panel figure composition |

Install all packages with:

```r
install.packages(c("haven", "fixest", "MatchIt", "data.table",
                   "rdrobust", "rddensity", "synthdid", "ggplot2",
                   "scales", "stringi", "patchwork"))
```

**LaTeX** packages used by `size_gradient_report.tex`: mathpazo, berasans, beramono, microtype, geometry, setspace, booktabs, array, threeparttable, siunitx, amsmath, graphicx, placeins, xcolor, hyperref, titlesec, natbib. All are included in standard TeX distributions.

## Data files

### Primary data shipped with the repo

**`data_raw/italy/electoral_panel_dataset.dta`** — Italian municipal electoral panel, 7,964 municipalities × 5 national elections (2001, 2006, 2008, 2013, 2018). Source: Cremaschi, Rettl, Cappelluti & De Vries (2024), Harvard Dataverse. DOI: [10.7910/DVN/I3VHZK](https://doi.org/10.7910/DVN/I3VHZK).

**`data_raw/italy/ministero_interno/unioni_comuni_ministero_2020.csv`** — Italian Ministry of the Interior registry of inter-municipal consortia (*unioni di comuni*), 2020 snapshot. Source: Italian Ministry of the Interior, *Anagrafe delle Unioni di Comuni*, [dait.interno.gov.it/territorio-e-autonomie-locali/sut/open_data.php](https://dait.interno.gov.it/territorio-e-autonomie-locali/sut/open_data.php).

**`data_raw/italy/opencivitas/Ind_FC20*.csv`** — OpenCivitas *Fabbisogni Standard* indicators by Article 14 fundamental function (police, waste, social services, territorial planning, road maintenance, civil registry, tax collection, education). Source: SOSE / OpenCivitas, 2015 reference year ([opencivitas.it](https://opencivitas.it/)). Used to construct binary indicators of associated service delivery.

**`data_raw/italy/cremaschi_replication/service_dataset.dta`** — Cremaschi et al.'s standardized service-output panel used to reproduce their Table 3 service-quality estimates (Figure 5). Source: Cremaschi, Rettl, Cappelluti & De Vries (2024), Harvard Dataverse. DOI: [10.7910/DVN/I3VHZK](https://doi.org/10.7910/DVN/I3VHZK).

### Derived data (regenerated by `make`)

- `data_processed/italy/electoral_panel_extended.csv` — main panel extended to include the 2022 general election. Built by `code/italy/05_extend_panel.R` from the original Cremaschi panel plus 2022 election results downloaded from the [onData project](https://github.com/ondata/elezioni-politiche-2022).
- `data_processed/italy/post2010_union_indicator.csv` — binary flag for post-2010 *unione* membership, built from the Ministry registry by `code/italy_mechanism/build_post2010_union_indicator.R`.
- `data_processed/italy/opencivitas_panel_crosswalk.csv` — match between OpenCivitas `USERNAME` codes and `id08` ISTAT municipality codes, built by `code/italy_mechanism/00_build_crosswalk.R`.
- French commune panel, downloaded from [data.gouv.fr](https://www.data.gouv.fr/fr/datasets/elections-presidentielles-resultats/) (election results) and [insee.fr](https://www.insee.fr/fr/statistiques/) (commune population and geography) by `code/france/00_download.R`.

## Pipeline

The `Makefile` orchestrates the full pipeline. Running `make` executes the steps below in dependency order. Inspect with `make -n` to see commands without running them.

| Stage | Scripts | Produces |
|---|---|---|
| Italy panel build | `code/italy/03_download_2022.R` → `04_build_2022.R` → `05_extend_panel.R` | `data_processed/italy/electoral_panel_extended.csv` (extends Cremaschi panel through 2022) |
| Italy main analysis | `code/italy/01_analysis_italy.R` | Italy tables (`tab_temporal`, `tab_gradient`, `tab_covariate_mediation`) and `placebo_scatter.pdf` |
| Italy figures | `code/italy/02_figure_italy_gradient.R`, `06_figure_bandwidth_sweep.R`, `07_figure_placebo_sweep.R` | `fig_logpop_facet.pdf`, `fig_bandwidth_sweep.pdf`, `fig_placebo_sweep.pdf` |
| France pipeline | `code/france/run_all.R` (chains scripts `00`–`05`) | `data_processed/france/final/panel_commune.csv`, `tab_temporal_fr.tex`, `fig_logpop_facet_fr.pdf`, `placebo_scatter_fr.pdf` |
| France sweeps | `code/france/06_figure_bandwidth_sweep_fr.R`, `07_figure_placebo_sweep_fr.R` | `fig_bandwidth_sweep_fr.pdf`, `fig_placebo_sweep_fr.pdf` |
| Italy mechanism (§4) | `code/italy_mechanism/build_post2010_union_indicator.R`, `00_build_crosswalk.R`, then `fig_union_formation_did.R`, `fig_compliance_gradient_no_rd.R`, `fig_service_diff_delta_no_rd.R`, `fig_compliance_gradient.R` | `fig_union_formation_did.pdf`, `fig_compliance_gradient_no_rd.pdf`, `fig_service_diff_delta_no_rd.pdf`, `tab_compliance_rd.tex` |
| LaTeX compile | `pdflatex` ×3 + `bibtex` (per document) | `size_gradient_report.pdf`, `size_gradient_report_appendix.pdf`, `size_gradient_report_with_appendix.pdf` |

## Data sources

- **Italian electoral panel:** Cremaschi, Rettl, Cappelluti & De Vries (2024). Replication Data for: *Geographies of Discontent*. Harvard Dataverse. DOI: [10.7910/DVN/I3VHZK](https://doi.org/10.7910/DVN/I3VHZK)
- **2022 Italian election results:** onData project, processed Eligendo files. [github.com/ondata/elezioni-politiche-2022](https://github.com/ondata/elezioni-politiche-2022)
- **Italian Ministry of the Interior, *Anagrafe delle Unioni di Comuni***: 2020 registry of inter-municipal consortia, with constitution dates and member municipalities. (Shipped in `data_raw/italy/ministero_interno/`.)
- **OpenCivitas / SOSE:** *Fabbisogni Standard* questionnaires (FC20 series), 2015 reference year. Records, for each municipality, whether each Article 14 fundamental function was delivered in associated form with neighbouring municipalities. [opencivitas.it](https://opencivitas.it/). (Shipped in `data_raw/italy/opencivitas/`.)
- **French presidential election results (2002–2022):** Ministère de l'Intérieur, aggregated elections dataset. [data.gouv.fr](https://www.data.gouv.fr/fr/datasets/elections-presidentielles-resultats/)
- **French commune population (2006 census):** INSEE, *populations légales*. [insee.fr](https://www.insee.fr/fr/statistiques/)
- **French commune geography (2026 COG):** INSEE, *Code Officiel Géographique*. [insee.fr](https://www.insee.fr/fr/information/6800675)
